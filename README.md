# Анализ развития игровой индустрии (2000–2013)

## Описание проекта

Данный исследовательский проект направлен на анализ и срез данных, касающихся развития игровой индустрии в период с 2000 по 2013 год. Цель проекта — провести предобработку данных, очистить их, заполнить пропуски и категоризировать информацию для выявления ключевых тенденций, таких как популярные платформы и общие показатели качества игр.

## Данные

Основным источником данных является файл `games_data.csv`, содержащий информацию о продажах, жанрах, платформах, а также оценках игр.

## Как запустить проект

Убедитесь, что у вас установлены все необходимые библиотеки, перечисленные в файле `requirements.txt`.

```bash
pip install -r requirements.txt
```

В проекте использовалось свое ядро Jupyter для работы с окружением и зависимостями из файла requirements.txt.
Для использования такого ядра нужно зарегистрировать это окружение в Jupyter под понятным именем, например:

```bash
python -m ipykernel install --user --name=my-venv-kernel --display-name "Python (My Project)"
```

## Краткий план анализа (в рамках ноутбука):

1.  **Загрузка и знакомство с данными:** Загрузка `games_data.csv` и проверка структуры.
2.  **Предобработка данных:**
    *   Нормализация названий столбцов (перевод в snake_case);
    *   Приведение типов данных (преобразование продаж и оценок в float);
    *   Обработка пропусков (восстановление лет выпуска по названию игр, обработка пропусков в оценках);
    *   Нормализация текстовых полей (регистр).
3.  **Анализ:** Фильтрация данных по диапазону лет (2000–2013) и категоризация.
4.  **Выводы:** Определение топ-платформ и общие выводы по рынку.
